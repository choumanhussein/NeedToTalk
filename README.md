## Execution Procedure
> $ git clone https://github.com/choumanhussein/NeedToTalk.git <br>
> $ cd NeedToTalk<br>
> $ bundle install<br>
> $ rails db:create && rails db:migrate<br>
> $ rails s<br>
> $ rails s -b 0.0.0.0 (Vagrant users)
